### 1. LLMs Fundamentos

- [ ] Zero-shot, Few-shot, Chain-of-Thought

- [ ] Técnicas avançadas (Tree of Thought, Self-Consistency)

- [ ] LangChain — chains, agents, tools

- [ ] Semantic Kernel (Microsoft)

### 6. Fine-tuning
